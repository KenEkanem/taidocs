# Submit Contact Message

Sends a contact message to support. Requires name, email, and message fields.

## Request Body (form-data)

| Field   | Type   | Required | Description                        |
|---------|--------|----------|------------------------------------|
| name    | string | ✅ Yes   | Full name of sender                |
| email   | string | ✅ Yes   | Email address                      |
| subject | string | ❌ No    | Optional subject                   |
| message | string | ✅ Yes   | Message body                       |

## Example Request

```bash
curl --location 'https://postman-rest-api-learner.glitch.me/api/v1/contact-message' \
--form 'name="Kazeem Asifat"' \
--form 'email="damkaz.ak@gmail.com"' \
--form 'subject="A simple message"' \
--form 'message="This is a sample contact us message, it will definitely work"'
```
