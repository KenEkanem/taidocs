# Get All Countries

Returns a list of countries with their ISO codes, flags, and phone codes.

## Example Request

```bash
curl --location 'https://postman-rest-api-learner.glitch.me/api/v1/countries'
```

## Response

```json
{
  "success": true,
  "message": "Countries retrieved successfully",
  "data": [
    {
      "id": 1,
      "iso2": "AD",
      "iso3": "AND",
      "flag": "https://flagcdn.com/ad.svg",
      "name": "Andorra",
      "phone_code": "376"
    }
  ]
}
```
