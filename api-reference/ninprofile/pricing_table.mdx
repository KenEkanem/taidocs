# Get Pricing Table

Returns a breakdown of pricing for verification and SSO actions.

## Example Request

```bash
curl --location 'https://postman-rest-api-learner.glitch.me/api/v1/pricing'
```

## Response

```json
{
  "success": true,
  "message": "Pricing breakdown retrieved successfully.",
  "data": [
    {
      "uuid": "083dd413-937e-43b7-b376-f0562bb7b76b",
      "name": "credit_rate",
      "value": "200",
      "display_name": "Amount per credit",
      "description": "1 credit costs 200 Naira"
    }
  ]
}
```
