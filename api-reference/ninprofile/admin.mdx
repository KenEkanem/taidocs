### POST - Change Status

`https://postman-rest-api-learner.glitch.me//api/v1/admin/user/update/:uuid`

_Update User Information_

This endpoint allows the admin to update user information by providing the user's UUID.

<Callout title="Authorization" type="warning">
Bearer Token: `{{admin_token}}`
</Callout>

#### Path Variables

- `uuid` – **required**

#### Body (form-data)

| Key    | Value     | Required | Description                     |
|--------|-----------|----------|---------------------------------|
| status | active/inactive | ✅       | Can be either `active` or `inactive` |

---

#### Example Request (cURL)

```bash
curl --location 'https://postman-rest-api-learner.glitch.me//api/v1/admin/user/update/:uuid' \
--form 'status="active"'
```
